# SchoolConnect v0.5

Web app designed to help students have a better access to their academic infos such as grades, attendance, resources etc...

This app was coded using MAMP first then XAMPP as local hosting support. It uses a SQL database accessed via PHP, though most of the variables are hard-coded for privacy.

This second design of the app was made by Arthur Grynblat, at the time responsible of User Experience.

<img width="1364" height="914" alt="image" src="https://github.com/user-attachments/assets/ccc94384-d5da-45f6-81ce-4cf395952cd4" />

As you can see the interface is light and simple, students have access to all they need through the home page, then they can access further infos through the other sections, accessible at the top.
At the bottom left of the screen you have the chat section, each class has its own dedicated class chat, making communications easier, and more respectful with its swear word filter. 

<img width="1360" height="912" alt="image" src="https://github.com/user-attachments/assets/55c1c171-7f9c-4e7e-8e44-885917bad9f6" />

The "Cours" section is where teachers have the possibility to upload a summary of the last lesson so missing students can still get the most important parts without having to ask friends for their notes.

# Possible improvements

Because of the project's cancellation due to a lack of clients, most of the features are not yet implemented, the teacher interface being the one lacking the most because our possible client wanted to focus more on the student experience than the teacher one.

Because the design was exported from Sketch using the anima toolkit, most of the CSS is redundant and each class has its own style rules, even though most of them have the same attributes. This makes for some weird bugs where everything is left aligned instead of center aligned because one rule is stepping over other ones.

This project is now 7 years old and has gone 5 years without being touched, the code is messy, some bits are inspired by Stack Overflow posts, and some bits are just useless.
